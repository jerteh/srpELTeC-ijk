Kako objaviti
  1. Kreirajte repo naziva username.github.io (zamenite username svojim GitHub korisničkim imenom).
  2. Dodajte ovaj fajl kao index.html u koren repozitorijuma i napravite Commit.
  3. Otvorite Settings → Pages i proverite da je Source: Deploy from a branch, grana main, folder / (root). Sačuvajte.
  4. Nakon par trenutaka, vaša stranica će biti dostupna na https://username.github.io.
